# Basic Operations Of verilog
> Executes Basic Verilog Operations

## Installation

You can copy this using this command:

```bash
git clone https://github.com/rksingh23/Find_Max.git
```

### Run main:

You can run this using .dat file provided`:

Double Click the do_CRC.bat file 

or execute the comment on command Prmpt
```.bat
vsim -do do_CRC.do
```

### Run Tests:
You need to have Questasim or Modelsim installed on your local host PC.
